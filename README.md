# Hybrid Quantum Circuit Simulator

This project is the IT Team Induction Task for the Phoenix Association (2025-26). It implements a basic hybrid quantum circuit simulator from scratch using Python and NumPy, designed to represent qubits, apply quantum gates, and simulate probability-based measurement outputs without relying on black-box quantum execution libraries like Qiskit.
